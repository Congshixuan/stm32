# STM32F103C8T6 流水灯模块化实现思路
**一、整体设计思路**

1. 分层分离：将流水灯驱动代码单独封装到 BSP 外设文件夹，与主业务 main.c 隔离。
2. 文件拆分规范
  ◦ .c文件：存放函数实体、变量定义（实现逻辑）
  ◦ .h文件：存放函数声明、全局变量外部声明，提供对外调用接口
3. 依赖逻辑
  ◦ 依赖stm32f1xx_hal.h：提供 GPIO 操作底层函数
  ◦ 依赖main.h：读取 CubeMX 配置的 LED 引脚宏（LED_R_Pin、LED_G_Pin、LED_B_Pin、GPIOA）
4. 硬件逻辑：低电平引脚输出灯亮（共阳极 RGB 灯），三色依次点亮延时循环，实现流水效果
5. 工程配置配套：Keil 添加文件分组、配置头文件搜索路径，使编译器能找到自定义驱动文件
